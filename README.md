# Open Termsheet

스타트업 투자 텀시트(Term Sheet)의 주요 조항을 **창업자와 투자자 양쪽 관점**에서 해석하는 오픈소스 가이드입니다.

https://open-termsheet.vercel.app

## 왜 만들었나

- 텀시트 조항은 법률 용어로 가득 차 있어 창업자가 이해하기 어렵습니다
- 투자자가 왜 그 조항을 넣는지 객관적으로 설명하는 자료가 부족합니다
- 한국 상법/벤처투자법 기준으로 정리된 자료가 거의 없습니다

## 특징

- **양면 해석** — 각 조항을 창업자/투자자 관점에서 설명
- **사례 기반** — 구체적인 숫자 시나리오로 조항의 실질적 영향을 보여줌
- **법률 검토** — 한국 상법, 벤처투자법, 벤처기업법 등 관련 법령과 리스크 포인트
- **시장 기준** — founder-friendly ↔ investor-friendly 스펙트럼 표시

## 정보 출처

본 프로젝트의 내용은 다음 자료를 기반으로 작성되었습니다.

- **대한민국 상법** — 종류주식(제344조), 전환주식(제346조), 자기주식 취득(제341조), 주식양도(제335조) 등
- **벤처투자 촉진에 관한 특별법 (벤처투자법)** — 벤처투자조합의 투자 조건(제37조), 투자금 회수(제50조) 등
- **벤처기업육성에 관한 특별조치법 (벤처기업법)** — 스톡옵션 특례(제16조의3) 등
- **자본시장과 금융투자업에 관한 법률 (자본시장법)** — IPO 심사 관련 규정
- **한국 벤처캐피탈 업계의 표준 투자계약 실무 관행** — RCPS(상환전환우선주) 기반 신주인수계약서 구조
- **모태펀드(한국벤처투자) 발간 「벤처기업을 위한 벤처캐피탈 투자계약서 해설서」** — 벤처투자법 적용 기준 및 투자 조항 해설
- **K-IFRS (한국채택국제회계기준)** — RCPS의 부채/자본 분류 관련 회계 기준

## 시작하기

```bash
npm install
npm run dev
```

http://localhost:3000 에서 확인할 수 있습니다.

## 기여하기

새로운 조항 추가, 기존 해석 보완, 법률 검토 업데이트 등 모든 기여를 환영합니다.

### 조항 추가 방법

1. `src/data/clauses/` 에 새 파일 생성
2. `Clause` 타입에 맞춰 데이터 작성 (`src/types/clause.ts` 참고)
3. `src/data/clauses/index.ts`에 등록
4. PR 제출

### 기여 시 참고사항

- 사례는 구체적인 숫자와 시나리오를 포함해주세요
- 법률 검토는 근거 법령(조문 번호)을 명시해주세요
- 특정 회사나 펀드의 내부 자료를 직접 인용하지 마세요 — 객관화된 형태로 작성해주세요
- 이 프로젝트는 법률 자문이 아닙니다 — 실제 계약 시 전문가 검토가 필요하다는 점을 항상 명시해주세요
